opencv.py
이미지의 크기를 줄여 사용되는 토큰의 낭비를 줄인다.
가공된 이미지는 "output.jpg" 으로 저장된다.

api.py
제미나이의 api를 사용해 이미지를 분류한다.
결과는 카테고리를 참고해 "대분류번호.소분류번호" 형식으로 나온다.
분류가 불가능한 이미지는 "18" 로 통일한다.
결과값이 저장된 파일 "sort_result.txt"를 이용해 분류한다.
